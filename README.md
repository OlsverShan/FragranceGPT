# FragranceGPT

**基于大语言模型的香水配方预测与推荐系统。** 给定 5 种主香调（例如，*柑橘调、木质调、芳香调、清新辛香调、琥珀调*），预测最有可能的前调 / 中调 / 后调 —— 并从 24,000+ 款真实香水中进行 RAG 检索。目前正在扩展到香水推荐和配方分析领域。

作为 **严谨的大语言模型评估** 案例研究构建：包含 7 种消融方法、双指标评估（词元 F1 + LLM-as-Judge）、模糊香调匹配以及诚实的负面结果。

## 消融实验结果（使用模糊匹配）

| 方法 | 精确 F1 | 模糊 F1 | Δ | 结论 |
|--------|----------|----------|---|---------|
| 随机猜测 | 0.006 | 0.006 | — | 下限 |
| 频率统计方法 | 0.299 | 0.327 | +9.4% | 基线 |
| 大语言模型零样本 | 0.337 | 0.393 | +16.6% | 基线 |
| 大语言模型 + 少样本 | 0.360 | — | — | 单独使用时有效 |
| **大语言模型 + RAG** | **0.406** | **0.443** | **+9.1%** | **最佳** |
| RAG + 少样本 | 0.405 | 0.442 | +9.1% | 冗余 |
| 多智能体协同 | 0.359 | 0.418 | +16.4% | 有害 |

各层级完整的精确率/召回率/F1 详细信息见 `results/ablation_summary.txt`。

### 关键发现：F1 分数不等于质量

LLM-as-Judge 评估显示，我们的预测结果在评分上常常超过真实标签：

| | 香调准确度 | 层次正确性 | 协调性 | 专业性 | **综合评分** |
|---|---|---|---|---|---|
| RAG 预测 | 4.37/5 | 4.13/5 | 4.37/5 | 4.37/5 | **4.37/5** |
| 真实标签 (Fragrantica) | 3.87/5 | 3.87/5 | 3.90/5 | 3.80/5 | **3.82/5** |

F1 与评判器之间的相关性 = -0.139。词元重叠度量的是记忆能力；LLM 评判器度量的则是泛化能力。

## 数据集

[Fragrantica.com 香水数据集](https://www.kaggle.com/datasets/olgagmiufana1/fragrantica-com-fragrance-dataset) — 包含 24,063 款香水、84 种香调、1,655 种独特香料。

## 关键要点

1. **RAG 是主导性技术** — 从领域数据库中动态检索能够提供大语言模型预训练所无法获得的上下文信息。
2. **并非每种技术都有帮助** — 少样本提示与 RAG 组合时是冗余的；多智能体分解对知识检索类任务有负面影响。
3. **真实标签未必是真实的** — LLM-as-Judge 揭示了众包标签存在噪声。一个好的系统可以超越其训练数据。
4. **模糊匹配修正了 F1 分数** — 通过规范化香料名称变体，所有方法的性能提升了 9-17%。
5. **双指标评估对于主观领域至关重要** — F1 追踪的是记忆能力；LLM 评判器追踪的是泛化能力。
